anatomyof.ai

This website explains how Alexa works. It explains both the hardware (the microphones that allow the device to hear the user) and the software (the AI agent that gauges how well Alexa responded to the user’s voice command). It then goes on to explain the materials required to make an Alexa device and the time and processes that create these materials (specifically lithium). It pulls sources from history to make the user consider the ideas behind Alexa and how these ideas have been translated across history to her final design.

endless.horse

This website shows a horse made through keyboard characters. Its legs will stretch as far down as the user scrolls.

thezen.zone

This website offers 3 forms of meditation, each with the option to do the activity once, three times, and five times. One, the swirl, has the user erase a spiral, another, has the user toggle a block of switches, and the final has the user hover their mouse above a set of 4 circles until they are smaller and smaller circles. After the user completes the activity, they are told how long they’ve been meditating for, though the “breaking” activity wildly overstates the amount of time spent on the activity.
